# AI Quiz Generator

An AI-powered quiz application built with Python, Streamlit, and Groq Llama 3.3 70B.

Users can enter any topic and instantly generate a quiz with multiple-choice questions.

## Features

* Dynamic quiz generation using AI
* User-defined quiz topics
* 5 multiple-choice questions
* Exactly 4 options per question
* Score tracking
* Progress indicator
* Quiz review page
* Responsive Streamlit interface

## Tech Stack

* Python
* Streamlit
* Groq API
* Llama 3.3 70B
* Python Dotenv

## Installation

1. Clone the repository

2. Install dependencies

pip install -r requirements.txt

3. Create a .env file

GROQ_API_KEY=your_api_key

4. Run the application

streamlit run app.py
