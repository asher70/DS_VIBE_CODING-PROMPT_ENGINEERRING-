# Habit Tracker

A simple Habit Tracker application built with Python, Streamlit, and SQLite.

## Features

* Create and manage habits
* Daily habit check-ins
* Streak tracking
* Progress tracking
* Calendar view
* Analytics dashboard
* SQLite database storage

## Tech Stack

* Python
* Streamlit
* SQLite
* Pandas
* Plotly

## Installation

Clone the repository:

git clone https://github.com/yourusername/habit-tracker.git

Navigate to the project folder:

cd habit-tracker

Install dependencies:

pip install -r requirements.txt

Run the application:

streamlit run app.py
