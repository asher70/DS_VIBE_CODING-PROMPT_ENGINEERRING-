# Expense Tracker

A Python Streamlit application for tracking income and expenses.

## Features

- Add income
- Add expenses
- Edit transactions
- Delete transactions
- Dashboard
- Charts and analytics
- CSV export
- Excel export

## Tech Stack

- Python
- Streamlit
- SQLite
- Pandas
- Plotly

## Installation(as per ai)

git clone <repo-url>

cd expense_tracker

pip install -r requirements.txt


streamlit run app.py
